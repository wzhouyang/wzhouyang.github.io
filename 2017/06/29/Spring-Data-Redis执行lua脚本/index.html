1. Redis 服务器lua脚本支持

redis从2.6.0版本开始支持lua脚本。其内置了一个lua解释器，可以对Lua5.1脚本程序解释执行。
redis提供了以下命令对Lua脚本进行支持：

1.1 EVAL

命令格式如下

  • EVAL第一个参数是一段Lua5.1脚本程序
  • EVAL第二个参数是参数的个数
  • EVAL从第三个参数开始表示在脚本中使用到的redis键（key）,这些键可以在脚本中通过全局变量KEYS数组获取（注意，lua数据下标从1开始）。
  • EVAL的arg [arg…]参数是不是键的附加参数，在脚本中可以通过全局变量ARGV的数组获取。

1.2 EVALSHA

命令格式如下：

根据给定的SHA1值，对缓存在服务器中的脚本进行执行。将脚本缓存到服务器的操作可以通过 SCRIPT LOAD 命令进行。

其他参数参见1.1 EVAL

1.3 SCRIPT EXISTS

命令格式：

这个命令可以接受一个或者多个脚本SHA1信息，返回一个1或者0的列表，如果脚本存在或不存在

1.4 SCRIPT FLUSH

清空Lua脚本缓存

1.5 SCRIPT LOAD

将脚本加载到服务器，但是并不执行，加载完成后即可调用1.2 EVALSHA执行
