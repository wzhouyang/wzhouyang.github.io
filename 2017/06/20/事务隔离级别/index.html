事务隔离级别

Jun 20, 2017 | Hits

1. 概要

数据库事务隔离级别有4个，由低到高分别为Read uncommitted（读未提交），Read committed（读已提交），Repeatable read（重复读），Serializable（序列化），四个级别可以逐个解决脏读，不可重复读，幻读问题。

2. 脏读，幻读，不可重复读

  1. 脏读

    脏读就是一个事务读取到了另一个事务中未提交的数据。

  2. 不可重复读

    不可重复读就是在一个事务内读取表中的某行数据，多次读取的结果不同。

    不可重复读与脏读的区别是：脏读读取了前一个事务未提交的数据，不可重复读则是读取到了前一个事务提交的数据。

  3. 幻读

    幻读就是指在一个事务中读取到了别人的事务插入的数据，导致前后数据不一致。

3. 事务隔离级别

级别 脏读 不可重复读 幻读
Read uncommitted Y Y Y
Read committed N Y Y
Repeatable read N N Y
Serializable N N N
