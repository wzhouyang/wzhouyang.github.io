1.lua类型

Jul 17, 2017 | Hits

1. 概述

Lua中有8种基础数据类型：

  1. nil(空)
  2. boolean(布尔)
  3. number(数字)
  4. string(字符串)
  5. userdata(自定义类型)
  6. function(函数)
  7. thread(线程)
  8. table(表)

type函数总是返回字符串。

2.1 nil

在Lua中nil是一种类型，用来表示无效的值。一个变量在未初始化前它的值就是nil，将一个
变量的值赋值为nil就等于删除了它。

2.2 boolean

boolean类型有两个值：true和false。在Lua中boolean却不是一个条件值的唯一
条件表达式，在Lua中任何值都可以表示一个ie条件，Lua将false和nil视作“假”，而将除此
之外的其他值视作“真”。因此在条件判断中，Lua将空字符串和数字零都视作“真”。

2.3 number

number类型用于表示实数。在Lua中没有整数类型，Lua中的数字可以完整的表示32位整数。
Lua中的数字支持科学计数法表示。

2.4 string

Lua中字符串表示“字符序列”。Lua采用8位编码表示字符串，Lua中的字符串可以具有任何数值编码。
在Lua中字符串是不可变的。

Lua的字符串支持转义字符。

在Lua中可以使用一对匹配的双放括号来表示字符串块。

如果在字符串中存在双方括号，则可以使用[====[那么Lua会在遇到相同数据量的等号的右方括号才会结束：

Lua提供了数字和字符串的自动转换，在一个字符串上应用算数运算符操作时Lua会尝试将其转换为数字。

2.5 table

table类型实现了关联数组，关联数组是一种具有特殊索引的数组;不仅可以通过整数来进行索引，
还可以使用字符串或者其他类型的值（nil除外）。

table没有固定大小，可以添加任意数量的元素到table中

2.6 function

Lua中function是作为第一类值来看待的，这表示函数可以存储在变量中来进行传递，返回等。
